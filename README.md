# ClearNano — Gemini Nano Banana 浮水印移除工具

ClearNano 是一款**純前端、零依賴**的網頁工具，專門用於移除 Google Gemini 生成圖片右下角的 Nano Banana 浮水印。所有運算完全在瀏覽器本地執行，圖片不會離開您的裝置。

---

## ✨ 功能特色

| 功能 | 說明 |
| :--- | :--- |
| 🔒 **純本地處理** | 圖片絕不上傳伺服器，完全在瀏覽器內完成 |
| 📐 **數學精確還原** | 反向 Alpha 混合算法（非 AI 補圖），精準恢復原始像素 |
| ⚡ **自動尺寸偵測** | 對應完整 Gemini 官方尺寸目錄（0.5k / 1k / 2k / 4k / Flash），自動選用正確遮罩與邊距 |
| 🧵 **Web Worker 加速** | 像素計算移出主執行緒，介面不卡頓 |
| 🚀 **並行批次處理** | 同時處理最多 4 張，大量圖片也能快速完成 |
| 🖼️ **輸出格式切換** | 預設輸出 JPEG（較小檔案），可切換為 PNG 保留透明度 |
| 👁️ **原圖 / 處理後對比預覽** | 點擊結果卡片，可即時切換查看前後差異 |
| 🌙 **深色 / 淺色 / 自動主題** | 介面主題可手動切換，或跟隨系統偏好自動套用 |

## 🚀 快速開始

### 前置需求

- [Node.js](https://nodejs.org/) 任意版本（僅用於啟動本地 HTTP 伺服器）

> 如果您已有其他本地 HTTP 伺服器工具（如 Python、nginx 等），也可以直接使用。

### 步驟

**1. 下載專案**

```bash
git clone https://github.com/YOUR_USERNAME/ClearNano.git
cd ClearNano
```

或直接在 GitHub 上點選 **Code → Download ZIP**，解壓後進入資料夾。

**2. 啟動本地伺服器**

```bash
# Windows：直接雙擊 run.bat
# 或在終端機輸入：
npx -y http-server -p 8080 -c-1
```

> `-c-1` 代表停用快取，確保每次重新整理都取得最新版本。

**3. 開啟瀏覽器**

前往 `http://127.0.0.1:8080`，即可看到 ClearNano 介面。

> ⚠️ 請勿直接用 `file://` 開啟 `index.html`，Web Worker 需要 HTTP 環境才能正常運作。

## 🖱️ 使用說明

### 上傳圖片

- **拖放**：將一張或多張 Gemini 生成圖片（PNG / JPG / WebP）拖入中央上傳區。
- **點擊選擇**：點擊上傳區，從檔案瀏覽器選取圖片。

### 選擇輸出格式

上傳區下方有格式切換按鈕：

- **JPEG**（預設）：檔案較小，適合一般分享用途。
- **PNG**：無損壓縮，保留透明通道。

### 查看結果

處理完成後，每張圖片會出現結果卡片，顯示：

- 縮圖預覽
- 偵測到的浮水印尺寸（logo size）與邊距（margin）
- 下載按鈕

**點擊卡片**可開啟全螢幕對比模式，切換「原圖」和「處理後」頁籤進行比較。

### 批次下載

點擊右上角「**下載全部**」按鈕，可一次下載所有處理結果。

### 切換介面主題

點擊頁面右上角的主題切換按鈕，選擇：

- ☀️ **淺色**
- 🌙 **深色**
- 🔄 **自動**（跟隨系統偏好，預設）

## 📐 浮水印偵測規則

ClearNano 採用完整的官方尺寸目錄進行精確比對，無法比對時則以空間相關性（Pearson 係數）自動推算最佳配置。

| 圖片尺寸範例 | Logo 大小 | 右邊距 | 下邊距 | 遮罩版本 |
| :--- | :---: | :---: | :---: | :--- |
| 0.5k 系列（如 512×512） | 48px | 32px | 32px | bg_48 |
| 1k 系列（如 1024×1024） | 48px | 32px | 32px | bg_48 |
| 2k 系列（如 2048×2048） | 96px | 64px | 64px | bg_96 |
| **2816×1536**（新邊距，2026-05 起） | 96px | 192px | 192px | bg_96_20260520 |
| 4k 系列（如 4096×4096） | 96px | 64px | 64px | bg_96 |
| Gemini 2.5 Flash 1k 系列 | 48px | 32px | 32px | bg_48 |

> 非標準尺寸圖片（如截圖、二次壓縮等）將自動投影至最近的官方尺寸配置進行處理。

## 🔬 技術原理

Gemini 浮水印的本質是一張具有特定 Alpha 值的白色圖層，疊加在原始影像右下角。

還原公式如下：

```
Pixel_original = (Pixel_final - α × 255) / (1 - α)
```

- `Pixel_final`：帶有浮水印的最終像素值
- `α`：遮罩 PNG 在該位置的 Alpha 值（0.0 ~ 1.0）
- `Pixel_original`：還原後的原始像素值

ClearNano 讀取對應的遮罩 PNG（`assets/bg_48.png`、`assets/bg_96.png`、`assets/bg_96_20260520.png`），逐像素套用此公式，計算由 Web Worker 非同步執行以避免介面凍結。

## 📁 專案結構

```
ClearNano/
├── index.html               # 主頁面
├── app.js                   # 核心邏輯（偵測、處理、UI）

└── workers/
    └── watermark-worker.js  # Web Worker（反向 Alpha 混合計算）
```

## ⚠️ 免責聲明

本工具會直接修改圖檔像素。雖然採用數學精確算法，仍可能因以下情況產生非預期結果：

- 圖片經過 JPEG 壓縮導致像素失真（JPEG artifacts）
- Gemini 浮水印規格更新
- 非官方尺寸或特殊格式圖片

**作者對任何資料遺失、圖片損壞或非預期修改不承擔任何責任，使用風險由使用者自行承擔。**
